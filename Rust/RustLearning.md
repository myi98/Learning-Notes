## Rust Installation

For Linux or Windows WSL
```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

For Windows

* Install Visual C++ 2013 up first
* Download RUSTUP-INIT.EXE and run
